# Technology Stack

## Language
- **Dart** (v3.x) — Primary language for Flutter development

## Framework
- **Flutter** (stable channel) — Cross-platform UI framework for Android & iOS

## State Management
- **setState / StatefulWidget** — Built-in state management for MVP simplicity

## Testing
- **flutter_test** — Flutter's built-in testing framework
- **integration_test** — For integration and end-to-end tests

## Build & CI
- **Gradle** (Android) — Native Android build system
- **Xcode / CocoaPods** (iOS) — Native iOS build system
- **Git** — Version control

## Target Platforms
- Android (minSdkVersion: 21+)
- iOS (minimum iOS version: 13+)

## Future Considerations (not for MVP)
- State management: Riverpod, Bloc, or Provider
- Networking: http, dio, or graphql_flutter
- Local storage: shared_preferences, sqflite, or hive
